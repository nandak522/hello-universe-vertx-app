### Building and running the app
```sh
docker compose build
docker compose up
```

### Building Vertx app alone

To launch your tests:
```sh
./gradlew clean test
```

To package your application:
```sh
./gradlew clean assemble
```

To run your application:
```sh
./gradlew clean run
```
